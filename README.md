# VBot Todo Android
## Yêu cầu
HBuilderX version 4.85 

Tải HbuilderX tại : https://www.dcloud.io/hbuilderx.html
---
## Cài đặt Dependencies

Clone Repository:

```text
git clone https://github.com/HoangHuuTinIT/vbot-todo-android.git
```
---
## Mở dự án
1. Mở HBuilderX
2. Trên thanh công cụ chọn file > Open Directory > Chọn thư mục chứa code repo > Select Folder
---
## Cấu hình biến môi trường
Dự án sử dụng các file .env để quản lý các biến môi trường cần thiết

### Các file cấu hình
.env: chứa các biến dùng chung

.env.example: file biến môi trường mẫu

### Tạo file cấu hình
1. Chuột phải vào tên dự án(thư mục gốc của dự án)
2. Chọn New > Custom File
3. Ở hộp thoại hiện lên, đặt tên file là .env
4. Copy toàn bộ nội dung của .env.example qua file .env vừa tạo
5. Thay giá trị thực tế cho các biến môi trường (như chú thích trong .env.example)
---
## Cài đặt các gói cần thiết:
1. Kết nối điện thoại với máy chạy code repo
2. Trên thanh công cụ chọn Run > Mobile app Playground > Android Run > Tại hộp thoại hiện lên chọn Run(O)
3. Trong quá trình chạy code, HBuilderX sẽ yêu cầu tải các thư viện cần thiết để chạy code , thông báo sẽ xuất hiện ở dưới cùng bên tay phải
4. Tải các thư viện mà HBuilderX yêu cầu.
